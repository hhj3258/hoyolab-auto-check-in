# HoyoLab 자동 출석체크

HoyoLab 출석체크를 자동으로 수행하는 스크립트입니다. 젠레스 존 제로, 붕괴: 스타레일, 원신, 미해결사건부, 붕괴3rd를 지원합니다.

**언어:** [English](README.md) | 한국어

---

## 요구사항

- [Python 3.8 이상](https://www.python.org/downloads/) — 실행 전 직접 설치 필요
- Windows

> 브라우저는 별도로 설치하지 않아도 됩니다. 스크립트가 자동으로 설치하고 관리합니다.

---

## 사용법

1. `run.bat` 실행
2. 최초 1회만:
   - 언어 선택 (한국어 / English / 日本語)
   - 출석체크할 게임 선택
   - 필요한 패키지 자동 설치
   - 브라우저가 열리면 HoyoLab에 로그인
3. 이후 실행부터는 백그라운드에서 자동으로 출석체크 수행

---

## 작업 스케줄러

최초 실행 시 매일 자동 실행 여부를 물어봅니다.
이후 등록/해제는 `schedule.bat`에서 처리할 수 있습니다.

> HoyoLab은 UTC+8 자정(KST 01:00)에 초기화됩니다. KST 01:05 이후 실행을 권장합니다.

**등록 확인:** `Win + S` → "작업 스케줄러" 검색 → 작업 스케줄러 라이브러리 → **HoyoLab 출석체크** 항목 확인

---

## 파일 구조

```
├── run.bat               # 실행 진입점
├── schedule.bat          # 작업 스케줄러 관리
├── locales/              # 다국어 문자열 (ko / en / ja)
└── scripts/
    ├── checkin.py        # 메인 스크립트
    ├── _setup.py         # 패키지 설치
    └── _schedule.py      # 작업 스케줄러 등록/해제
```

## 동작 방식

### 브라우저 자동화
[Playwright](https://playwright.dev/python/)를 사용해 Chromium 브라우저를 백그라운드에서 실행합니다.
로그인 세션이 로컬에 저장되므로 로그인은 최초 1회만 필요합니다.

### 로그인 감지
브라우저가 열린 후 로그인 완료 여부를 자동으로 감지합니다.
감지되면 세션이 저장되고 브라우저가 자동으로 닫힙니다.

### 출석체크
1. 선택한 게임별 HoyoLab 출석체크 페이지를 엽니다
2. 이번 달 완료 횟수를 읽어 다음 체크인 일차를 결정합니다
3. API 응답을 통해 오늘 이미 완료된 경우 별도 조작 없이 즉시 종료합니다
4. 대상 일차가 초기 표시 범위를 벗어난 경우 "더 보기" 버튼을 클릭합니다
5. 보상 카드를 클릭하고 완료 여부를 확인합니다

### 세션 만료
저장된 세션이 만료된 경우 브라우저를 자동으로 열어 재로그인 후 출석체크를 재시도합니다.

## 보안 및 개인정보

- **계정 정보를 저장하지 않습니다.**
  - 스크립트는 HoyoLab 아이디나 비밀번호를 읽거나 저장하지 않습니다.
  - 로그인은 실제 브라우저 창에서 공식 HoyoLab 사이트를 통해 직접 진행됩니다.
- **세션 데이터는 내 PC에만 저장됩니다.**
  - 로그인 세션(쿠키)은 `data/browser_profile/` 폴더에 로컬로 저장되며 외부로 전송되지 않습니다.
- **외부 서버를 거치지 않습니다.**
  - 스크립트는 HoyoLab과 직접 통신하며, 일반적인 웹사이트 이용과 동일합니다.
- **오픈소스입니다.**
  - 이 리포지토리에서 모든 코드를 직접 확인할 수 있습니다.

## 설정 변경

`data/.lang` 파일을 삭제하면 다음 실행 시 언어 선택 화면이 다시 나타납니다.
`data/.games` 파일을 삭제하면 다음 실행 시 게임 선택 화면이 다시 나타납니다.
